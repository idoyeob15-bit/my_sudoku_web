# my_sudoku_web
🧩 Sudoku Rival 실시간 멀티플레이로 즐기는 긴장감 넘치는 스도쿠 대결 앱입니다.  ✨ 주요 기능  실시간 대결: Socket.IO 기반으로 상대방의 정답 현황을 실시간 확인.  초대 시스템: 닉네임 입력만으로 간편하게 친구와 매칭.  하트 시스템: 3번 오답 시 즉시 패배! 신중한 수 읽기가 핵심.  가독성 특화: 3x3 구역 강조 디자인으로 보드판을 한눈에 파악.  🎮 플레이 방법  닉네임 설정 후 접속.  상대방 ID로 초대장 발송 및 수락.  하트가 다 닳기 전에 먼저 보드를 완성하면 승리!
